Templates
=========

These YAML configuration files can be copied or downloaded and used as templates for various image analysis task, such as the detection of objects (with or without masking areas, reference dectection and measurement tasks) or for setting landmarks. Here you can also download the configuration files used in the `phenopype gallery <https://www.phenopype.org/gallery>`_ If you would like to download all available templates, `click here <https://raw.githubusercontent.com/phenopype/phenopype-templates/main>`_ to download the entire phenopype template repository.
