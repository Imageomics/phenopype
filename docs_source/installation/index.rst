Installation
============

Brief instructions
------------------

Create a Python3 (v3.7) environment (e.g. using conda):

.. code-block:: bash

	conda create -n "pp" python=3.7

Activate the environment:

.. code-block:: bash

	conda activate pp

Install phenopype:

.. code-block:: bash

	pip install phenopype



Detailed instructions
---------------------

These instructions are intended for Python beginners and assume that :code:`Anaconda` is being used to manage Python and :code:`pip` for the phenopype installation. If you don't know what this means or are in doubt about what to do, just stick to the following instructions.
